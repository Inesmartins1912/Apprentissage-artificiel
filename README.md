# Projet Final d'Apprentissage Artificiel

## Objectif

Ce projet est une reproduction de la tâche 3 du DÉfi Fouille de Texte (DEFT) de 2009 : apprentissage de classification par parti politique d’interventions au parlement européen. C'est à dire qu'on vise à prédire le parti politique des locuteurs à partir de leur discours. Il s'agit plus précisément d'une tâche de classification de texte par parti politique.   

L'objectif est ici d'explorer et de comparer les performances de différents modèles d'apprentissage automatique classiques (SVM, Naive Bayes, Random Forest) et de rechercher comment améliorer leurs performances.   

## Données

Les données sont disponibles sur le site de [DEFT](https://deft.lisn.upsaclay.fr/), leur description et celle de la tâche dans les [actes](https://deft.lisn.upsaclay.fr/actes/2009/pdf/0_grouin.pdf).
